# DPS

Publisher-Subscriber (Pub-Sub) protocols are widely spread in nowadays technology. Because of their flexibility they could be easily implemented in a vast set of heterogeneous environments. Today this kind of implementations rely on centralized infrastructures, based on broker services and databases hosted on servers.
